# 호텔 예약 시스템
> 간단한 호텔 예약시스템을 구축함. 주요하게 생각해야 할 점은 호텔 방에 따라 예약 정원이 있으며, 정원이 가득찰 경우 
> 예약을 하지 못함. 동시성 에슈에 대해 생각해야함

## 정리


## SPEC
* java 17
* spring-boot 3.2.5
* jpa
* redis
* eureka-server
* api-gateway

## 동시성 고민
* 2가지 방법
  * 낙관적 락 : 버저닝 사용, 제일 간단
  * 비관적 락 : 데이터 수정동안 접근을 제한함
    * Redisson 을 사용해서 비관적 락 사용
